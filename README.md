# Primer API
A través del uso del framework de NestJs logré crear mi primer API, con la metodología CRUD(Create, Read, Update, Delete).

Utilizando TypeScript. Además de:
- Apache
- Postman
- MySQL

## Description

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
